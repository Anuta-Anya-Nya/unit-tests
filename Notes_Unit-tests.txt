Тестирование - написание кода для проверки лругого кода
программная ошибка - ошибка, из-за которой программа выдает неожиданное поведение
    синтаксические ошибки устраняются сразу. на моменте компиляции
    логические  - не обнаруживаются сразу 
    выполнения - не оптимизированный алгоритм, исключения 

ПРинципы тестирования:
тстирование показывает наличие дефектов - тестирование снижает появление дефектовю если ошибки не найдены, это не означает, что их нет, а если найдены, то что они найдены все
исчерпывающее тестирование невозможно - слишком много входных данных, поэтому делается выборка
раннее тестирование - неодходимо начинать тестирование как можно раньше
кластеризация дефектов - 80% ошибок содержатся в 20% модулей
парадокс пестицидов - если долго делать одни и те же тесты, через долгое времяя они не будут выявлять дефекты
тестирование зависит от контекста - новостной портал тестируется иначе нежели банковское приложение
заблуждение в отсутствии ошибок - 

жизненный цикл ПО 
анализ-планирование-проектирование-разработка-тестирование-развертывание- снова анализ 
    анализ - формируются бизнес требования к продукту
    планирование - расчет затрат, сбор команды
    проектирование - моделирование ПО, обсуждение архитектуры, создание прототипа.
    разработка - написание программы
    тестирование 
    разветывание - ПО доступно конечному пользователю

Ошибки проверяют в пограничных случаях 

тесты принято выносить в отдельный класс
throw new AssertionError непроверенное исключение
throw new ArithmeticException арифметическое исключение
throw new IllegalstateException 

утверждение (assert) - встроенный в java механизм проверки правильности предположений.
по умолчанию утверждения отключены, их надо запускать с флагом -ea (java -ea .\CalculatorTest.java)
утверждения не исключают исключений
утверждения проверяют только булевы функции
утверждения использовать, кога проверка полезна, но помнить что они могут быть отклчены и не отработать
исклчения надо использоваь если после получения исключения программа должна дольше как-то отраотать 

AssertJ - библотека java, которая предоставляет набор утверждений и полезных сообщений об оштбках, улучшает читаемость кода
